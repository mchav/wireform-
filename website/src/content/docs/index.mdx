## Quick start

```bash
git clone https://github.com/iand675/wireform-.git && cd wireform-
cabal update && cabal run example-msgpack
```

Three lines to a working round-trip. No `.proto` file, no codegen, no external tooling.

## Unified by design

Every format package follows the same conventions. Define your type once,
derive for as many formats as you need:

```haskell
data Person = Person { name :: !Text, age :: !Int }
  deriving stock (Show, Eq, Generic)
  deriving anyclass (ToMsgPack, FromMsgPack, ToCBOR, FromCBOR, ToYAML, FromYAML)
```

Same patterns, same vocabulary, same `Value` types — whether you're working
with MessagePack or Parquet.
