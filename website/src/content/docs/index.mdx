## What is wireform?

wireform is one Haskell library that ships **encode/decode**, **schema parsing**,
**code generation**, **streaming**, **RPC framing**, **container I/O**, and
**analytics metadata** under one consistent surface — across 25+ wire formats.

<CardGrid stagger>
  <Card title="One library, many formats" icon="seti:default">
    Protocol Buffers, Avro, Thrift, MessagePack, CBOR, BSON, XML, HTML,
    Parquet, ORC, Arrow, Iceberg, FlatBuffers, Cap'n Proto, ASN.1, Bond,
    EDN, Ion, TOML, NDJSON, CSV, Bencode, plus the wireform core.
  </Card>
  <Card title="Schema-aware codegen" icon="rocket">
    A single CLI (`wireform-gen`) generates idiomatic Haskell from
    `.proto`, `.avsc`, `.thrift`, `.bond`, `.fbs`, `.capnp`, `.asn1`,
    `.xsd`, and CDDL/ISL files.
  </Card>
  <Card title="Performance discipline" icon="setting">
    Unboxed sums on the decoder hot path, `withTag` CPS dispatch,
    `Data.Text.Builder` everywhere, no `String` round-trips. See the
    benchmark suite in `bench/`.
  </Card>
  <Card title="Plays well with others" icon="puzzle">
    Decode one format's `Value`, map across, re-encode in another —
    wireform's `Value` types share the same conventions across formats
    so format conversion is a few lines.
  </Card>
</CardGrid>

## Try it now

```bash
git clone https://github.com/iand675/wireform-.git && cd wireform-
cabal update
cabal run example-msgpack     # Generics-derived MessagePack — quickest demo
cabal run example-basic       # hand-written protobuf message
cabal run example-xml         # Generic XML encode/decode
```
