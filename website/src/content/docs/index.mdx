## Quick start

```bash
git clone https://github.com/iand675/wireform-.git && cd wireform-
cabal update && cabal run example-msgpack
```

Three lines to a working round-trip. No `.proto` file, no codegen, no external tooling.

## Capabilities at a glance

<CardGrid>
  <Card title="Encode / Decode" icon="open-book">
    Type-safe serialization for every supported format, with `Generic` deriving
    for schema-less formats and codegen for schema-first workflows.
  </Card>
  <Card title="Streaming" icon="right-arrow">
    Incremental decoders for Protocol Buffers, MessagePack, CBOR, and XML.
    Feed chunks as they arrive — no buffering the full payload.
  </Card>
  <Card title="RPC framing" icon="puzzle">
    gRPC, Thrift binary/compact, MsgPack-RPC, and Avro IPC framing
    built in — no separate transport library needed.
  </Card>
  <Card title="Analytics formats" icon="document">
    Read Parquet pages, ORC stripes, Arrow IPC batches, and Iceberg
    manifests — without heavyweight JVM tooling.
  </Card>
</CardGrid>
