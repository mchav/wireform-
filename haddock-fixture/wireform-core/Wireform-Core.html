Wireform.Core

Description

The umbrella module of wireform-core. Re-exports the pieces every backend needs: the wire Decoder, the encode-side Builder, and the schema-tagged Value type.

Format-specific modules live under Wireform.Proto, Wireform.Avro, etc.

The decoder monad

data Decoder a Source

The core wire decoder. Wraps a strict ByteString + offset and returns Either DecodeError a with the new offset.

Decoders are strict; there is no laziness on the success path.

runDecoder :: Decoder a -> ByteString -> Either DecodeError a Source

Run a Decoder on a complete ByteString, returning either the decoded value or the first error encountered.

Schema-tagged values

data Value Source

A schema-tagged value. Every wireform format can decode into Value and encode out of it; format conversion is a few lines of Map manipulation.
